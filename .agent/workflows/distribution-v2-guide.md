# 📦 Distribution V2 คืออะไร? (อธิบายแบบง่ายๆ)

ลองนึกภาพว่าเราเป็น **หัวหน้าทีม (Supervisor)** ที่ต้อง **แจกรายชื่อลูกค้า** ให้พนักงานไปโทรขาย หน้า Distribution V2 นี่แหละคือ **"หน้าจอแจกงาน"** ที่ใช้ทำเรื่องนี้!

---

## 🎯 มันคืออะไรกันแน่?

Distribution V2 เปรียบเหมือน **"ห้องแบ่งงาน"** ที่หัวหน้าเอาลูกค้าจาก "กองกลาง" มาแจกให้พนักงานแต่ละคน

**ตัวอย่างง่ายๆ:**
- เหมือนกับมีกล่องลูกอมใหญ่ (กองกลาง)
- หัวหน้าแบ่งลูกอมให้ลูกน้องคนละเท่าๆ กัน
- ลูกน้องแต่ละคนก็เอาไปขาย

---

## 🗂️ ถังแจกงาน (Distribution Baskets) มีอะไรบ้าง?

นี่คือ **"กองกลาง"** ที่มีลูกค้ารอแจก:

| ถัง | ความหมาย | แจกเข้าถังไหนของ Telesale |
|-----|----------|---------------------------|
| 📦 **Upsell** | ลูกค้าที่สั่งของล่าสุด | ถัง Upsell |
| 📦 **New Customer** | ลูกค้าใหม่ | ถังลูกค้าใหม่ |
| 📦 **ค้นหาเจ้าของใหม่** | ลูกค้าที่รอคนดูแล | ถัง 1-2 เดือน |
| 📦 **รอจับคู่** | ลูกค้าที่รอจัดสรร | ถัง Last Chance |
| 📦 **6-12 เดือน** | ไม่สั่งมา 6-12 เดือน | ถัง 6-12 เดือน |
| 📦 **1-3 ปี** | ไม่สั่งมา 1-3 ปี | ถัง 1-3 ปี |
| 📦 **Ancient** | ไม่สั่งมานานมาก | ถัง Ancient |

```

│ Upsell  │ │ New Cus │ │ ค้นหา   │ │ รวมทั้ง │

- **การ์ดสีเขียว** = รวมทั้งหมด (ดูได้อย่างเดียว)

```

**อธิบาย:**
- **☑ Checkbox** = เลือก/ไม่เลือกพนักงานคนนี้
- **ดึงคืน** = ดึงลูกค้าคืนจากพนักงานคนนี้ (ดูหัวข้อ "ดึงคืน")
- **ทั้งหมด** = จำนวนลูกค้าที่พนักงานถืออยู่
- **คอลัมน์ถัง** = จำนวนในแต่ละถังของพนักงาน

## 🚀 วิธีแจกงาน (Step-by-Step)

### ขั้นตอนที่ 1: เลือกถังที่ต้องการแจก
```
กดที่การ์ดถังด้านบน หรือเลือกจาก dropdown
เช่น "New Customer" หรือ "ค้นหาเจ้าของใหม่"
```

### ขั้นตอนที่ 2: กำหนดจำนวน
```
พิมพ์จำนวนลูกค้าที่ต้องการแจก
เช่น 100 คน (ต้องไม่เกินจำนวนที่มี)
```

### ขั้นตอนที่ 3: เลือกพนักงาน
```
✓ ติ๊กเลือกพนักงานที่จะแจกให้
  - เลือกได้หลายคน
  - ระบบจะคำนวณให้ว่าคนละกี่คน (≈ 25/คน)
```

### ขั้นตอนที่ 4: ดูตัวอย่างก่อนแจก
```
กดปุ่ม "👁️ ดูตัวอย่างก่อนแจก"
→ จะเห็นว่าใครจะได้ลูกค้าคนไหนบ้าง
```

### ขั้นตอนที่ 5: ยืนยันแจกงาน
```
ถ้าโอเค กด "✓ ยืนยันแจกงาน"
→ ลูกค้าจะถูกส่งไปยังถังของพนักงานแต่ละคน
```

## 🔙 การดึงคืนลูกค้า (Reclaim)

บางทีต้อง **เอาลูกค้าคืน** จากพนักงาน เช่น:
- พนักงานลาออก
- ต้องการปรับสมดุลงาน
- พนักงานมีงานเยอะเกินไป

### วิธีดึงคืน:

1. **กดปุ่ม "ดึงคืน"** ที่แถวของพนักงาน

2. **เลือกจำนวนจากแต่ละถัง:**
```
┌──────────────────────────────────────────────────────┐
│  ดึงลูกค้าคืนจาก สมชาย มาก                           │
├──────────────────────────────────────────────────────┤
│ ลูกค้าใหม่          │ มีอยู่ 30 │ [  10  ] │ [ทั้งหมด] │
│ 1-2 เดือน          │ มีอยู่ 25 │ [   5  ] │ [ทั้งหมด] │
│ Last Chance        │ มีอยู่ 20 │ [   0  ] │ [ทั้งหมด] │
│ ...                                                   │
├──────────────────────────────────────────────────────┤
│              [ ยกเลิก ]  [ ✓ ยืนยันดึงคืน ]           │
└──────────────────────────────────────────────────────┘
```

3. **กด "ยืนยันดึงคืน"**
   - ลูกค้าจะถูกส่งกลับไปยังถังแจกงาน
   - พร้อมแจกให้คนอื่นอีกครั้ง

## 🧠 Smart Allocation (ระบบแจกอัจฉริยะ)

ระบบมี **"ความฉลาด"** ในการแจกลูกค้า:

### ✅ สิ่งที่ระบบทำให้อัตโนมัติ:

1. **ไม่แจกซ้ำคนเดิม**
   - ถ้าลูกค้าเคยถูกแจกให้พนักงาน A แล้ว
   - รอบหน้าจะไม่แจกให้พนักงาน A อีก

2. **แจกเท่าๆ กัน**
   - เลือก 4 คน แจก 100 คน = คนละ 25 คน
   - พยายามให้เท่ากันที่สุด

3. **แจกตามลำดับความใหม่**
   - ลูกค้าที่สั่งล่าสุดจะถูกแจกก่อน
   - (ความสำคัญสูงกว่า)

### ⚠️ คำเตือนที่อาจเจอ:

| ข้อความ | ความหมาย |
|---------|----------|
| "บางพนักงานได้น้อยกว่ากำหนด" | มีลูกค้าซ้ำ ระบบข้ามไป |
| "ไม่มีลูกค้าพร้อมแจก" | ถังว่างแล้ว |
| "กรุณาเลือกพนักงาน" | ยังไม่ได้ติ๊กเลือกใคร |

## ❓ FAQ (คำถามที่พบบ่อย)

**Q: ทำไมบางถังกดดึงคืนไม่ได้?**
> A: ถัง Upsell ดึงคืนไม่ได้ และถังที่ไม่มี linked basket ก็ดึงไม่ได้

**Q: แจกไปแล้วยกเลิกได้ไหม?**
> A: ได้! ใช้ปุ่ม "ดึงคืน" เพื่อเอาลูกค้ากลับมา

**Q: ทำไมพนักงานได้ไม่เท่ากัน?**
> A: ระบบจะข้ามลูกค้าที่เคยแจกให้คนเดิมแล้ว (ไม่แจกซ้ำ)

**Q: ลูกค้าที่ดึงคืนไปไหน?**
> A: กลับไปที่ถังแจกงานที่ linked กับถังนั้น
