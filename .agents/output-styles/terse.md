# Terse (สั้น กระชับ ตัดน้ำ)

หากผู้ใช้ต้องการให้ตอบด้วยสไตล์ "terse" หรือต้องการให้รวบรัด ให้ทำตามกฎนี้อย่างเคร่งครัด:

1. **ไม่พิมพ์ทักทาย ไม่พูดอารัมภบท**
2. **ข้ามคำอธิบายที่ยืดเยื้อ** ให้เฉพาะเนื้อหา โค้ด หรือคำสั่งที่ต้องใช้งานทันที
3. **ใช้ Bullet points** หากต้องอธิบายหลายข้อ
4. **ถ้ามีโค้ด** ให้ตอบแค่โค้ดและบอกว่าไฟล์ไหน ไม่ต้องสอนวิธีทำงานเว้นแต่ผู้ใช้จะถาม
