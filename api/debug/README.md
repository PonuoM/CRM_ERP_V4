# Debug Scripts

โฟลเดอร์นี้เก็บไฟล์ debug scripts ที่ใช้สำหรับตรวจสอบและแก้ไขปัญหาในระหว่าง development

## ไฟล์ในโฟลเดอร์นี้:

- `debug_activities.php` - ตรวจสอบ schema ตาราง activities
- `debug_customers.php` - ตรวจสอบ schema ตาราง customers
- `debug_detection.php` - ตรวจสอบ detection logic
- `debug_function.php` - ตรวจสอบ database functions
- `debug_orders_data.php` - ตรวจสอบข้อมูล orders
- `debug_orders.php` - ตรวจสอบ schema ตาราง orders
- `debug_others.php` - ตรวจสอบอื่นๆ
- `debug_remaining.php` - ตรวจสอบ remaining data
- `debug_schema_v2.php` - ตรวจสอบ schema version 2
- `debug_tags.php` - ตรวจสอบ schema ตาราง tags
- `debug_triggers.php` - ตรวจสอบ database triggers
- `debug_triggers_v2.php` - ตรวจสอบ database triggers version 2

## หมายเหตุ:

ไฟล์เหล่านี้ **ไม่จำเป็นสำหรับ production** และไม่ได้ถูกใช้ใน production code
เก็บไว้สำหรับ development และ debugging เท่านั้น
