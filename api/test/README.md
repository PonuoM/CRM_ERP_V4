# Test Scripts

โฟลเดอร์นี้เก็บไฟล์ test scripts ที่ใช้สำหรับทดสอบการทำงานของระบบ

## ไฟล์ในโฟลเดอร์นี้:

- `test_customer_update.php` - ทดสอบการอัพเดท customer
- `test_duplicate.php` - ทดสอบ duplicate detection

## หมายเหตุ:

ไฟล์เหล่านี้ **ไม่จำเป็นสำหรับ production** และไม่ได้ถูกใช้ใน production code
เก็บไว้สำหรับ development และ testing เท่านั้น
